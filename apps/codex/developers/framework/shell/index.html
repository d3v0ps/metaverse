Offline-first Micro-frontends Management Shell based on [Angular](https://angular.io/)

  • Modules managed with Nx and documented with Storybook and Compodoc.
  • Web Components based on SCSS and CSS variables themed through Style Dictionaries. is used for documentation.
  • Dependencies injected through Angular in the FrontEnd side and NestJs in the BackEnd side.
  • Offline-first Entity Manager and Repositories built on top of RxDB with query observables and ACL.
  • Desktop native with Electron and Electron Build.
  • Tabs, Iframes and Webpack Module Federation apps support.
  • 6495 Icons from Material Design Icons and 2135 Simple Icons.
  • Keyboard shortcuts with Hotkeys and Gamepad API.
